# Hostel Management Application

Created using MERN Stack with Redux

## Installation

```
npm install
cd Reactapp_frontend && npm install
```

### Configure .env file

```
NODE_ENV = development/production
PORT = 5000
MONGO_URI= Mongo Uri
JWT_SECRET= JWT Key
```

## Features

- [x] Register/Login Screens
- [x] Student Details and Student data manipulation
- [x] Student attendance
- [x] Download Csv file of attendance
- [x] Delete Attendance of previous n days
- [x] Control User List
- [x] Edit User Admin Status
- [x] Data Controlled by only Admins
